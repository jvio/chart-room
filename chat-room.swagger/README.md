# Skeleton project for Swagger

#Prerequisites

This project is based on swagger node:

https://github.com/swagger-api/swagger-node/

```
npm install -g swagger
```

## Edit

```
npm run edit
```

## Run

```
npm start
```

## Test

```
npm run test
```
